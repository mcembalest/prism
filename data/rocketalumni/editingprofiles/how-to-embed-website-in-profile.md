1. Click on the the profile you want to embed a website in

2. Go the settings tab (first tab)

3. Click on the entry layout options and once the popup opens, scroll to the embedded website layouts

4. After selecting the layout, go to the text tab (second tab)

5. Input the website’s URL to be embedded

Note: Some URLs may cause issues when trying to embed them. Here are some things to try if you can’t embed a website:

HTTP vs HTTPS: Websites that begin with http cannot be embedded as iFrames. This is because they are not secure. Websites that begin with https can in most instances be embedded as iFrames. The exception to this is where the original website does not allow itself to be embedded. 

Content Security Policy headers: you might need to specify CSP headers that allow you to specify what parent source may embed a page. Sending this article to your school’s IT department may be helpful.
